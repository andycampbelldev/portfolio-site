Doctor Manilowe

This is a site I built for a social media meme account. I had a lot of fun designing and building this one to fit Doctor Manilowe's back story, and I particularly enjoyed researching and trying to replicate early 2000s web design while ensuring the site was still fully responsive.

  • 2000s-inspired responsive frontend built from scratch with plain old HTML, CSS and JS
  • Hosted on GitHub Pages
  • Simple quiz implemented with client-side JS
  • Contact form implemented with Formspree
