Doctor Manilowe

Doctor Manilowe... is a weird, long story. In short, this is a website I built for a meme account centered around vegan mayonnaise. I had a lot of fun designing and building this one to fit Doctor Manilowe's back story, and I particularly enjoyed researching and trying to replicate early 2000s web design.

Have a look through Doctor Manilowe's website for yourself - test your knowledge on the quiz or send Doctor Manilowe a nice message via the Contact Page!

For the unfamiliar, here's the Doctor Manilowe Notion Wiki for some much needed context!

  • 2000s-inspired responsive frontend built from scratch with plain old HTML, CSS and JS
  • Deployed to GitHub Pages with custom domain
  • Simple quiz implemented with client-side JS
  • Contact form implemented with Formspree
