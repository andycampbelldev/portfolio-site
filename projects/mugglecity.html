Muggle City

A home hub for storing and quickly accessing favorite recipes, shopping lists and other vital home info for ourselves and guests - like how to use the coffee machine.

  • Front-end built with vanilla HTML, CSS and JavaScript
  • NodeJS and Express backend
  • Simple data persistence using NodeJS file system methods and JSON file storage
  • Basic CMS for creating custom pages with basic HTML and image upload to Cloudinary
  • Gmail integration for emailing shopping list
