Muggle City

One of my first self-guided projects. A home hub for storing and quickly accessing favorite recipes, shopping lists and other vital home info for ourselves and guests - like how to use the coffee machine.

  • Responsive frontend built with plain old HTML, CSS and JavaScript
  • Server-side EJS templating
  • NodeJS and Express backend
  • Data persistence using NodeJS file system methods and JSON file storage
  • CMS for creating custom pages with HTML and image upload to Cloudinary
  • Nodemailer for emailing shopping list
